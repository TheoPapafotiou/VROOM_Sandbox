# VROOM_Sandbox
This is a repository for all testing files developed by student team VROOM in Aristotle University of Thessaloniki.. 

## How to enable pre-commit hooks
You need to have python and pip installed on your computer

### Linux: <br>
```
$ python3 -m venv venv
$ source venv/bin/active
$ pip install -r requirements.txt
```
#### Windows: <br>
*OS is deprecated you should use linux instead*
```
$ pip install virtualenv
$ virtualenv env
$ \env\Scripts\activate
$ pip install -r requirements.txt
```
